## Usage

One time usage,

    $ npx blame-my-network http://theanubhav.com

    output:
    Connection successful from both, internal and external network.

Installing global package

    $ npm i -g blame-my-network
    $ blamemynetwork http://theanubhav.com
    Connection successful from both, internal and external network.

    output:
    Connection successful from both, internal and external network.
